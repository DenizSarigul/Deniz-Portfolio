Informations sur ce projet

Période : Du 04/10/2024 au 15/11/2024

Langage utilisé : Java

Travail : Personnel

Introduction : Ce projet est une application Java avec pour architecture MVC, ce dernier sera éxécuté dans l’environnement de développement (IDE Eclipse). Ce projet a pour but une gestion d'une bibliothèque de jeux vidéo (editeurs,catégories,consoles,plateformes,jeux) en faisant du CRUD (Create,Read,Update,Delete).

Bilan personnel : Ce projet m'a permis de pouvoir développer davantage mes compétences Java car j'avais des difficultées, mais aussi j'ai pu m'améliorer aussi en MVC. Sachant que c'est un projet solo,je suis plutôt satisfait de ce projet en Java.
